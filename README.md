# Manufacturing Analysis & OEE Prediction Tool

A data-driven manufacturing analytics and prediction system designed to evaluate machine performance, identify downtime patterns, and assist in production planning using machine learning.

---

## Project Overview

This project focuses on analyzing production and downtime data to:

- Compute **Overall Equipment Effectiveness (OEE)**
- Identify **downtime patterns and root causes**
- Analyze **machine and shift-wise performance**
- Build a **predictive model** for estimating machine performance
- Enable **data-backed scheduling decisions**
